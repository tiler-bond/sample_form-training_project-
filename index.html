This is a project for illustrating different array methods

array.push()-- The push() method adds a new element to an array (at the end)

array.pop()-- The pop() method removes the last element from an array

array.toString()-- The JavaScript method toString() converts an array to a string of (comma separated) array values.

array.join()-- The join() method also joins all array elements into a string but with a specified separator

array.shift()-- The shift() method removes the first array element

array.unshift()-- The unshift() method adds a new element to an array (at the beginning)

array.concat()-- The concat() method creates a new array by merging (concatenating) existing arrays

array.slice()-- The slice() method slices out a piece of an array into a new array.

array.splice()-- The splice() method can be used to add new items to an array or remove elements without leaving "holes" in the array`
